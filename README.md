# 速通AI编程开发：基于Roo Code+DeepSeek的编程教程

## 项目背景

很多接触到ai领域的初学者对于“ai编程开发”具有很大兴趣，但是又担心自己基础薄弱，难以入门，加上ai编程开发涉及到的工具和概念很多，很难快速上手。

本项目面向想要快速上手ai编程开发的初学者，通过Roo Code和DeepSeek的结合，帮助初学者快速上手ai编程开发。
